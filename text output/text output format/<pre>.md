# pre tag in PHP.
## meaning

    " <pre> " refers the text for output will be prefixed.
    
    That is to say, the format for text of output will NEVER be changed.
    
For example,

the original format of print_r() function looks like this.
    
    Array
    (
        [0] => 1
        [1] => 2
        [2] => 3
    )
